--- title: Javascript Libraries layout: post date: 2013-07-27 02:15:19 ---

Lately there have been a bunch of javascript libraries that I've been wanting to explore. Here are a few that look useful:

  • Popcorn.js

    This library is for interacting with multimedia, videos especially. You can set up events that fire when an embedded youtube video reaches certain points or when it is paused or played, things like that. When the video reaches the scary part, dim the page background to set the mood. When the video is paused, show an annoying ad. Imagine the possibilities...

  • Three.js

    3D rendering in the browser. Easy to follow syntax. Hardware acceleration capabilities. Gosh.

  • Hammer.js

    Code that supports touch gestures. Multitouch, drag, pinch, etc. I've been making some data visualizations that require the mouse for hovering and interaction. I think that if I applied gesture/touch events to the elements in the charts, fingers could take the place of a cursor.

Hopefully I can use more libraries like these to produce some enjoyable interactive web experiences
